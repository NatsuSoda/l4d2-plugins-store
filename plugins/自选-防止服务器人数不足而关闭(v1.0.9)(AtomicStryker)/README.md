## 简介

防止服务器人数不足而关闭

## 可用指令

- `sm_findentitybyclass` 通过类名和起始索引查找实体（管理员指令）
- `sm_findentitybyname` 通过名称和起始索引查找实体（管理员指令）
- `sm_listentities` 在服务器控制台输出所有有效实体（管理员指令）
- `sm_findnearentities` 查找你周围指定半径内的所有实体（管理员指令）
- `sm_sendentityinput` 向指定实体发送输入指令（管理员指令）
- `sm_findentprop` 返回你自身的实体属性值（管理员指令）
- `sm_findentmodel` 返回指定实体的模型路径（管理员指令）
