## 简介

Neko多特管理员菜单

## 可用指令

- `sm_ntg` 全功能特感菜单 **核心**
- `sm_nekotg` 全功能特感菜单
- `sm_tgmenu` 全功能特感菜单
- `sm_tgadmin` 全功能特感菜单

- `sm_ntgversion` 特感版本查询/状态检查
- `sm_nhud` HUD显示调整 (插件商店没有hud部分)
- `sm_reloadntgconfig` 重载多特配置文件(Neko_Specials_binhooks.cfg)
- `sm_reloadhudconfig` 重载HUD配置文件(Neko_KillHud_binhooks.cfg)(插件商店没有hud部分)
- `sm_updateservername` 手动更新服务器名字
- `sm_nekoupdate` 执行检查更新
- `sm_tgvoteadmin` 控制玩家投票模块
