## 简介

自定义击中和击杀的图标、声音、时长。
原版模式需要客户端vpk支持。
新版命中反馈，玩家可以使用`!hm` 切合不同的模式。

## 可用指令

- `sm_hitmarker` 开关命中反馈
- `sm_hitmarkers` 开关命中反馈
- `sm_hm` 开关命中反馈 **核心** 通常选择原版即可
- `sm_bhm` 开关命中反馈
- `sm_hmarker` 开关命中反馈
- `sm_crosshair_green` 更换准星颜色为绿色
- `sm_crosshair_green_change` 更换反馈变色时准星颜色为绿色
- `sm_crosshair_red` 更换准星颜色为红色
- `sm_crosshair_red_change` 更换反馈变色时准星颜色为红色
- `sm_crosshair_blue` 更换准星颜色为蓝色
- `sm_crosshair_blue_change` 更换反馈变色时准星颜色为蓝色
