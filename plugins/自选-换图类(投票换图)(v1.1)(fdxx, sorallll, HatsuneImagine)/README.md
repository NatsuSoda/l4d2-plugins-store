## 简介

换图类(投票换图)

## 可用指令

- `sm_v3` 投票换图
- `sm_maps` 投票换图
- `sm_chmap` 投票换图 **核心**
- `sm_vmap` 投票换图
- `sm_votemap` 投票换图
- `sm_mapvote` 投票切换下一张地图
- `sm_mapnext` 投票切换下一张地图 **核心**
- `sm_votenext` 投票切换下一张地图
- `sm_reload_vpk` 重新加载VPK地图文件（管理员指令）
- `sm_update_vpk` 更新VPK地图文件（管理员指令） **核心**
- `sm_missions_export` 导出战役列表（管理员指令）
